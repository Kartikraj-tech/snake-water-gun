# Snake Water Gun Game

A simple Snake Water Gun game built with Python.

## About
This is a beginner Python mini project based on the classic Snake, Water, Gun game.  
The user plays against the computer, and the winner is decided based on game rules.

## Features
- User vs computer gameplay
- Random computer choice using Python
- Input handling with dictionaries
- Beginner-friendly Python logic

## Technologies Used
- Python

## How to Run
1. Make sure Python is installed on your system.
2. Clone this repository or download the project files.
3. Open the project folder.
4. Run the Python file:

```bash
python main.py
